## [CentOS](http://centos.org) v6.7

* x86_64
* base install
* has VBoxGuestAdditions v5.0.8
* [Packer template](https://github.com/lgbarn/packer-templates/)

## Use

```
#
# Create initial Vagrantfile
#

vagrant init lgbarn/cent.6.7-x86_64

#
# Customize
#

vi Vagrantfile

#
# Start the box
#

vagrant up
```
