## [CentOS](http://centos.org) v6.5

* x86_64
* base install
* has VBoxGuestAdditions v5.0.8
* [Packer template](https://github.com/maier/packer-templates/)

## Use

```
#
# Create initial Vagrantfile
#

vagrant init maier/centos-6.5-x86_64

#
# Customize
#

vi Vagrantfile

#
# Start the box
#

vagrant up
```
